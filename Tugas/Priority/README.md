Langkah Percobaan :
1. Persiapan Alat dan Bahan
   - ESP32-S3 Board
   - OLED
   - LED
   - Buzzer
   - Push Button
   - Potentiometer
   - Encoder
   - Motor Stepper

2. Tujuan Percobaan :
   - Mempelajari cara pembuatan dan pengaturan beberapa task (FreeRTOS Task) pada ESP32 yang dijalankan secara paralel di dua core prosesor.
   - Menentukan dan menguji pengaruh prioritas (task priority) terhadap waktu eksekusi dan performa masing-masing perangkat.

3. Pengujian
   - Upload program ke ESP32-S3.
   - Buka Serial Monitor.
   - Ubah Priority Task dan amati.
